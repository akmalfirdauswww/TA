# Urban Animal - Cat Disease Diagnosis

This is a web-based application for diagnosing cat diseases using the Gemini API. The application is built with Next.js and features a modern, minimalist UI.

## Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm, yarn, or pnpm

### Setup

1.  **Clone the repository:**

    ```bash
    git clone <repository-url>
    cd urban-animal-cat-diagnosis
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  **Set up your environment variables:**

    Create a file named `.env.local` in the root of the project and add your Gemini API key:

    ```
    GEMINI_API_KEY=your_gemini_api_key_here
    ```

    **Note:** You need to create the `.env.local` file manually as it is included in `.gitignore` for security reasons.

4.  **Run the development server:**

    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
